{\bfseries{GLOverlay\+UI}} is a desktop application that demonstrates the integration of a real-\/time Open\+GL rendering pipeline with a native wx\+Widgets GUI. It features an interactive 2D scene rendered on a {\ttfamily wx\+GLCanvas}, including a clickable Open\+GL overlay and a dynamic control panel using native UI widgets.

\begin{DoxyItemize}
\item ✅ Open\+GL-\/powered 2D scene rendered on {\ttfamily wx\+GLCanvas}
\item 🎯 Clickable UI overlay Geometric forms rendered directly in Open\+GL
\begin{DoxyItemize}
\item Using Mouse motion to control object rotation
\end{DoxyItemize}
\item 📌 Toggleable right-\/side control panel with\+:
\begin{DoxyItemize}
\item {\ttfamily wx\+Slider} to control object scale
\item {\ttfamily wx\+Check\+Box} to toggle visibility of scene objects
\item 🧼 Clean separation between rendering engine and UI logic
\item 🔁 Real-\/time interaction between Open\+GL scene and native UI
